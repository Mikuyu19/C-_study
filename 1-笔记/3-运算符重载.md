## 友元 friend
所谓友元 是指类与函数 或 类与类 之间的一种关系。
友元授权予某些函数或某些类，允许它们直接访问本类的隐藏信息。
根据友元的特性，可以分友元函数和友元类。

- 友元是普通的全局函数
如果友元是普通的全局函数，表示该函数可以直接访问当前类的隐藏信息，只需要在类中做友元声明，一般如下：
```
class 类名
{
    friend 返回类型 函数名(参数列表);
};
```
例：
```cpp
class Demo
{
public:
    // 全局函数的友元声明
    friend void print_demo(Demo d);
    Demo(int x = 0): x(x) {}

private:
    int x;
};

void print_demo(Demo d)
{
    cout << "d.x = " << d.x << endl;
}
```

- 友元是其他类的成员函数
如果友元是其他类的成员函数，表示该函数可以直接访问当前类的隐藏信息，只需要在类中做友元声明，一般如下：
```
class 类名
{
    friend 返回类型 其他类名::函数名(参数列表);
};
```

- 友元是其他类
如果友元是其他类，表示该类中所有的成员函数都能直接访问当前类的隐藏信息，只需要在类中做友元声明，一般如下：
```
class 类名
{
    friend class 其他类名;
};
```

说明：
1、友元声明可以出现在类中的任意位置。
2、全局友元函数可以类中直接定义，但它不是成员函数。
3、友元关系是单向的, 如果需要双方互为友元，则双方都得做出声明。
4、友元关系是不可传递的，如A是B的朋友，B是C的朋友，A和C不是朋友。
5、同类型的对象，互为友元。




## 运算符重载基本理论
所谓运算符重载，就是给已有的运算符添加新功能，使得这些运算符可以适用于类类型的对象。
在C++中，运算符是用函数实现的，所以，运算符重载的本质就是 函数重载。

运算符与运算符函数的对应规则：
1、运算符号 对应 运算符函数名称
    运算符函数名称由 关键字`operator`加上运算符号 组成
    例：
    << 对应的函数名称为  operator<<
    +  对应的函数名称为  operator+

2、操作数 对应 函数参数
3、运算过程 对应 函数体
4、运算结果 对应 函数返回值

说明：运算符函数编写完成后，一般由编译器自动调用(也可以像普通函数一样的调用)。


## 典型双目运算符
算术运算符： + - *  /
关系运算符: == !=  >  >=  <  <= 
这些运算符的重载方式相同，有两种方式：
1、可以重载为 全局函数, 一般需要做友元声明，这种函数有两个参数。
2、可以重载为 类的非静态成员函数,不需要友元声明，这种函数只有一个参数(左操作数是通过this传递的，右操作数做为参数)
注：这些运算符函数的形参，可以传值，也可以传const引用

例：
```cpp
class Demo 
{
public:
    // 全局函数的友元声明
    friend Demo operator+(Demo d1, Demo d2);
    Demo(int x=0): _x(x) {}
    void print() const 
    {
        cout << "Demo(" << this->_x << ")" << endl;
    }

    // ==号重载为非静态成员函数
    bool operator==(Demo d2) const 
    {
        if (this->_x == d2._x)
            return true;
        return false;
    }

private:
    int _x;
};

// +号运算符重载为全局函数
Demo operator+(Demo d1, Demo d2)
{
    int x = d1._x + d2._x;
    Demo d3(x);
    return d3;
}
```

## 输入输出运算符 
    <<  
    >>
IO流对象(cout/cin/cerr)不支持拷贝(拷贝构造函数被删除了)
所有，在函数传参和函数返回时，只能使用IO流对象的左值引用。

输入输出运算符 只能重载为 全局函数。

输出运算符的一般形式：
```
std::ostream& operator<<(std::ostream& out, 类名 对象名);
std::ostream& operator<<(std::ostream& out, const 类名& 引用名);
```

输入运算符的一般形式：
```
std::istream& operator>>(std::istream& in, 类名& 引用名);
```
例：
```cpp
class Demo 
{
public:
    // 友元声明
    friend std::ostream& operator<<(std::ostream& out, const Demo& d1);
    Demo(int x=0): _x(x) {}

private:
    int _x;
};

// 全局函数
std::ostream& operator<<(std::ostream& out, const Demo& d1)
{
    out << "Demo(" << d1._x << ")" ;
    return out;
}

int main(int argc, char const *argv[])
{
    Demo d1(100);
    cout << d1 << endl; 
    return 0;
}
```

## 下标运算符 []
[]只能重载为 类的非静态成员函数。
一般需要写两个版本：
只读
```
返回类型 operator[](int index) const ;
```
可读可写
```
返回类型& operator[](int index)  ;
```
例：
```cpp
class Demo
{
public:
    Demo()
    {
        p = new int[5]{1,2,3,4,5};
    }
    ~Demo()
    {
        delete [] p;
    }

    // 可读可写
    int& operator[](int index)  //  Demo* const this
    {
        return p[index];
    }
    // 只读
    int operator[](int index) const //  const Demo* const this
    {
        return p[index];
    }

private:
    int* p;
};
```

## 赋值运算符 =
对象的拷贝分两种：
拷贝构造，用一个已存在的对象，初始化一个【新】对象
拷贝赋值，用一个已存在的对象，给另一个【已存在】对象 赋值。
例：
```cpp
Demo d1;
Demo d2 = d1;   // 拷贝构造
d2 = d1;        // 拷贝赋值
```
如果类中没有显示的定义赋值运算符函数，编译器会自动生成。
赋值运算符 只能重载为 类的非静态成员函数。
一般如下：
```
class 类名
{
public:
    类名& operator=(const 类名&);
};
```
如果对象占用了系统资源，自动生成的赋值运算符就 可能出错，此时，就需要自定义赋值运算符。
一般如下：
1、防止自赋值
2、自定义拷贝过程
3、返回自引用


## 自增运算符 ++
自增运算符 有两种重载方式：
1、全局函数
2、类的非静态成员函数

前++的一般形式：
```
类名& operator++(类名&);    // 全局函数
类名& operator++();         // 非静态成员函数
```
后++的一般形式：
```
类名 operator++(类名&, int);    // 全局函数
类名 operator++(int);         // 非静态成员函数
```

说明：
    后++函数中的形参int，称为占位符参数，只占位置，不参与运算，用于区分前后++
    --与++类似。
例：
```cpp
class Demo
{
public:
    // 前++的友元声明
    friend Demo &operator++(Demo &d);
    // 后++的友元声明
    friend Demo operator++(Demo& d, int);
    Demo(int x = 0) : x(x) {}
    void print() const
    {
        cout << "Demo(" << x << ")" << endl;
    }

    // 前--重载为成员函数
    Demo &operator--()
    {
        --this->x;
        return *this;
    }

private:
    int x;
};

// 前++重载为全局函数
Demo &operator++(Demo &d)
{
    ++d.x;
    return d;
}

// 后++重载为全局函数
Demo operator++(Demo& d, int)
{
    Demo tmp = d;
    ++d.x;
    return tmp;
}
```


## 函数调用运算符 ()
()只能重载为 类的非静态成员函数。
一般如下：
```
class 类名
{
public:
    返回类型 operator()(参数列表)
    {
        函数体
    }
};
```
如果一个类重载了 函数调用运算符，这种类型的对象，可称之为 函数对象(仿函数)
函数对象的行为 类似于 函数，可以调用。


## 匿名函数(lambda表达式)(C++11)
lambda表达式 是一段可调用的代码单元。本质上是函数对象，也称为匿名函数。
语法格式：
`[捕获列表](参数列表)->返回类型{函数体}`
说明：
    参数列表，与普通函数的参数列表类似，但不能设置默认值，如果不需要传递参数，可以空着，也可以省略()
    返回类型，与普通函数的返回类型类似，但通常省略不写(包括->)，由函数体中的return语句自动推导。如果推导出来的数据类型与期望的不一致，可以手动显示的指定。
    函数体，可以为空，但{}不能省略。
    捕获列表，用于说明 lambda表达式 的函数体中需要使用的外界变量。
        如果需要捕获外界变量的值，在[]中写`变量名`即可，也可以用`=`来捕获所有外界变量的值，如果捕获的是变量的值，则只读。
        如果需要捕获外界变量的引用，在[]中写`&变量名`即可，也可以用`&`来捕获所有外界变量的引用，如果捕获的是变量的引用，则可读可写。
        如果不需要捕获外界变量的值，[]可以空着，但不能省略。
    

例：
```cpp
int main(int argc, char const *argv[])
{
    int x = 1;
    int y = 2;

    // 定义匿名函数，捕获局部变量当前的值，无参数，不返回，原地调用
    [=]() -> void
    {
        cout << "x = " << x << endl;
        cout << "y = " << y << endl;
    }();

    // 定义匿名函数，捕获局部变量的引用，一个参数，不返回，原地调用，调用时传参
    [&](int z)
    {
        x += z;
        y += z;
    }(100);

    cout << "x = " << x << endl;
    cout << "y = " << y << endl;

    // 定义匿名函数，不捕获局部变量的值或引用，无参数，不返回，使用auto命名
    auto print = []{
        cout << "hello lambda!" << endl;
    };

    // 通过名字调用lambda表达式
    print();

    return 0;
}
```

auto关键字说明：
    auto关键字从C++11起，不再表示自动变量，而是用于 根据对象的初值来自动推导对象的类型。
例：
```cpp
auto i = 1;         // int i = 1;
auto j = 1.2;       // double j = 1.2;
```

练习：
    给定一个整数数组，从该数组中查找 满足条件的第一个元素，找到则返回其下标，否则 返回-1
    要求用普通函数、函数对象、匿名函数 分别实现。
    条件示例：第一个3的倍数，第一个大于0的数等


## 对象的移动(C++11)
对象的移动也分两种：
移动构造，把一个已存在对象的资源，转移给同类型的新对象。
移动赋值，把一个已存在对象的资源，转移给另一个已存在的同类型对象。
例：
```cpp
void swap(string& s1, string& s2)
{
    string tmp = s1;    // 拷贝构造
    s1 = s2;            // 拷贝赋值
    s2 = tmp;           // 拷贝赋值
}
```

C++11增加了对象的移动操作，用于移动对象的资源。
为了实现移动操作，C++11引入了【右值引用】
右值引用，指只能绑定到右值的引用：
`类型名 && 引用名 = 右值;`
例：
```cpp
int a = 100;
int & r1 = a;           // 左值引用
const int& r2 = a;      // const引用绑定到左值
const int& r3 = 100;    // const引用绑定到右值
int && r4 = 100;        // 右值引用
```
如果一个构造函数的参数 是本类类型的右值引用，则这种构造函数，称为 移动构造函数。
如果一个赋值运算符函数的参数 是本类类型的右值引用，则这种赋值运算符，称为 移动赋值运算符。
一般如下：
```cpp
class 类名
{
public:
    类名(类名&&);
    类名& operator=(类名&&);
};
```
移动操作函数 不会申请新的资源，只需要转移已有的资源即可。
例：
```cpp
class Demo 
{
public:
    Demo(const char* s)
    {
        p = new char[strlen(s)+1];
        strcpy(p, s);
    }
    ~Demo()
    {
        delete[] p;
    }

    Demo(const Demo& rhs)
    {
        cout << "拷贝构造" << endl;
        p = new char[strlen(rhs.p)+1];
        strcpy(p, rhs.p);
    }
    Demo& operator=(const Demo& rhs)
    {
        cout << "拷贝赋值" << endl;
        if (this == &rhs)
            return *this;

        delete[]p;
        p = new char[strlen(rhs.p)+1];
        strcpy(p, rhs.p);

        return *this;
    }

    Demo(Demo&& rhs)
    {
        cout << "移动构造" << endl;
        p = rhs.p;
        rhs.p = nullptr;
    }
    Demo& operator=( Demo&& rhs)
    {
        cout << "移动赋值" << endl;
        if (this == &rhs)
            return *this;

        delete[]p;
        p = rhs.p;
        rhs.p = nullptr;
       
        return *this;
    }
private:
    char* p;
};
```

一般情况下，只有参数是右值(或右值引用), 才能触发移动操作函数。
为了显示的使用移动操作函数，C++标准库中提供了一个特殊的函数：move
move函数用于获取给定对象的右值引用。
典型示例：
```cpp
void swap(Demo& s1, Demo& s2)
{
    Demo tmp = std::move(s1);      // 移动构造
    s1 = std::move(s2);            // 移动赋值
    s2 = std::move(tmp);           // 移动赋值
}
```
当使用移动操作函数，把一个对象的资源转移给另一个对象后，原对象就不再拥有资源了。
对于原对象，要么重新给资源，要么不再使用。
